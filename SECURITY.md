# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately using GitHub's private advisory system:

**[Report a vulnerability](https://github.com/FelipeFuhr/ffreis-siteops/security/advisories/new)**

You will receive a response within 7 days. If the issue is confirmed, a fix will be released as soon as possible.
